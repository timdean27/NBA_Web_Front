1. Install Redux and Redux Toolkit (Best Practice):
Redux Toolkit simplifies the setup and usage of Redux. It provides createSlice for reducing boilerplate code and createAsyncThunk for handling asynchronous logic (like API calls).

Install the required packages:

npm install @reduxjs/toolkit react-redux axios
